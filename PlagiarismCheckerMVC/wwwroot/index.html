Поиск плагиата в Word-документах

PlagChecker - это сервис, который поможет вам проверить ваши документы на наличие плагиата с использованием мощных поисковых систем.

Возможности:

  • Проверка документов Word формата .docx
  • Использование API Google и Яндекс для поиска
  • Разбивка текста на абзацы и предложения
  • Удаление стоп-слов для повышения точности
  • Подробные отчеты с указанием источников заимствования
